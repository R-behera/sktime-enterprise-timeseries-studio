# Runbook

## Target user

Network operations and field service leaders

## Core operating loop

1. Load or bootstrap sample data.
2. Start the FastAPI application.
3. Use the web interface to score, analyze, and draft the next action.
4. Review health status before demo or deployment.

## Operator note

The main value of this repo is the workflow it supports: Pre-position crews and prioritize regions before service degradation worsens.. Keep the UI, docs, and API aligned with that business action.
